# C# Wrapper library for Signalsmith Stretch

This is a C# wrapper for the [Signalsmith Stretch](https://github.com/Signalsmith-Audio/signalsmith-stretch) library, which provides high-quality time-stretching and pitch-shifting capabilities.

Documentation: https://signalsmith-audio.co.uk/code/stretch/

Support .NET Framework 4.6.2 and .NET 8 or above.

## Example
There a example project in the `example` folder demonstrating usage of the library using miniaudio.
